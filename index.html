The colors in CSS

CSS supports an incredible variety of colors. It permits us to controls them even through number, this allow us to have infinite choices.
The colors in CSS can be written in 3 types:

  • Colors with name: you can use English words to describe colors. These are called "Keyword Colors" and we have, more or less, 140 of them.
  • RGB colors: numerical values that recall a color based to the mix of Red, Green and Blue.
  • HSL: numerical values that describe tint, saturation and lightness.

We have to make a distinction between:

  • Foreground color: color of the element itself. We can change it with the property 'color'.
  • Background color: color of the background of the choosed element. We can change it with the property 'background-color'.

RGB Colors

This is another type of syntax, we can use only numbers.Values starts from 0 to 255. These colors are the same of the hexadecimal system, but they are written in a different way. It's a matter of personal taste, but is a good practice to choose one of them and be coherent with it. RGB system is convenient because is more or less that the computers use to represent the colors. Here we have a few example of this type of syntax:

rgb(127,255,212) This is an example
rgb(220,20,60) This is an example
rgb(1,50,32) This is an example
rgb(255,255,240) This is an example

As you can see we use the same colors as above, but written in a different way. And the result is the same.

Hexadecimal Colors

This is another type of system called Hexadecimal.
An HEX color starts with an hash sign (#) followed by 3 or 6 characters. This characters represent values of red, green and blue. There are both numbers and letters, in fact this system have 15 characters instead of 9 like the standard numerical system. To represent numbers from 10 to 15, we can use letters form A to F.
These are a few examples:

#9FE2BF This is an example
#FF7F50 This is an example
#CCCCFF This is an example
#DFFF00 This is an example
